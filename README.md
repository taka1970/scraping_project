# 📦 Amazon Price Tracker

Amazon 商品の価格を自動で取得・保存・可視化する Python プロジェクトです。  
Playwright を使用して Amazon の商品ページをスクレイピングし、  
価格データを CSV に蓄積し、グラフとして可視化できます。

---

## 🚀 Features

- 🔍 Amazon 商品ページの価格を自動取得  
- 📁 日付ごとに価格を CSV に保存  
- 📊 価格推移をグラフ化（matplotlib）  
- 🧩 Playwright による安定したスクレイピング  
- 🕒 タスクスケジューラで毎日自動実行可能  
- 🧹 venv + .gitignore によるクリーンな構成
